# Todo List (Create React App)

Проект реализует все части задания в одном CRA-приложении с 3 режимами:

1. `JSONPlaceholder`:
только вывод списка `todos` (без CRUD).

2. `JSON Server`:
CRUD, поиск по фрагменту текста, debounce-поиск, переключаемая сортировка A-Z.

3. `Firebase`:
CRUD в Realtime Database, поиск и сортировка на клиенте.
