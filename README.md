# Todo List (React Router + JSON Server)

Версия для нового ДЗ (без Firebase), реализованная на базе предыдущего проекта.

## Функционал

### `/`
- список задач;
- добавление новой задачи;
- поиск по фразе (с debounce);
- переключаемая сортировка A-Z;
- карточка задачи в одну строку с многоточием (`...`) при переполнении;
- по клику на текст открывается страница задачи.

### `/task/:id`
- полное описание задачи;
- кнопка `Назад` (возврат на предыдущую страницу);
- редактирование задачи;
- удаление задачи.

### `/404`
- страница ошибки 404;
- переход по любому некорректному адресу ведет на `/404`.

## Запуск

1. Установить зависимости:
```bash
npm install
```

2. Запустить JSON Server:
```bash
npm run server
```

3. В отдельном терминале запустить React:
```bash
npm start
```

Открыть:
`http://localhost:3000`

## API

JSON Server стартует на:
`http://127.0.0.1:3001`

Endpoint:
`http://127.0.0.1:3001/todos`
